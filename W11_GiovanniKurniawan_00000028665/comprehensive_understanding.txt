(NOMOR 1)
site_url() => mengembalikan situs URL sesuai dengan yang ditentukan file konfigurasi.
anchor() => membuat tautan anchor HTML standar berdasarkan alamat URL lokal.	
mailto() => membuat tautan e-mail HTML standar.
redirect() => mengarahkan user dari halaman satu ke halaman lainnya.

(NOMOR 2)
